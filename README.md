Jacktoberfest 2016
==================
Repositorio para practicar Java, rellenando los métodos de las clases.
Realiza 4 PRs y consigue tu camiseta del Hacktoberfest :jack_o_lantern:.


## Contribuir

Haced **un solo PR** por cada  método y/o issue. Así podrá aportar más gente.

Por favor, abstenerse **expertos**, este repositorio pretende ser una ayuda 
para que programadores novatos puedan realizar los 4 PRs del Hacktoberfest y
ganar una camiseta.
